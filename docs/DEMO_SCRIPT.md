# 节点化演化画布 - 演示脚本

## 🎬 演示场景：创建宇航员演化链

### 场景 1: 创建初始节点
1. 打开 CineFlow 画布
2. 双击空白处创建图片节点
3. 输入提示词："A cute astronaut floating in space"
4. 点击生成
5. 等待图片生成完成

### 场景 2: 图片演化 - 添加动作
1. 悬停在宇航员节点上
2. 观察四个锚点出现（上下左右）
3. 点击并拖拽右侧锚点
4. 看到渐变连线跟随鼠标移动
5. 松开鼠标，弹出创建菜单
6. 选择 "✨ 变换图片"
7. 生成面板自动打开，参考图已填充
8. 修改提示词："Same astronaut, but running on the moon"
9. 点击生成
10. 子节点自动创建在右侧，并连线到父节点

### 场景 3: 图片转视频
1. 悬停在跑步宇航员节点上
2. 拖拽右侧锚点
3. 选择 "🎬 生成视频"
4. 视频生成面板打开
5. 选择运镜方向："镜头拉近"
6. 设置视频时长：6秒
7. 点击生成
8. 视频节点创建，显示生成进度

### 场景 4: 添加创作备注
1. 悬停在任意节点上
2. 拖拽下方锚点
3. 选择 "📝 添加备注"
4. 文本节点立即创建
5. 输入备注："这个镜头需要强调动感"
